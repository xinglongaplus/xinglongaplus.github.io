# 興隆A+ 青創團隊官網

## 📁 資料夾結構

```
xinlong-aplus-site/
├── index.html          ← 首頁（含開場動畫）
├── assets/
│   ├── logo-vertical.png    ← 直式 Logo（請自行放入）
│   └── logo-horizontal.png  ← 橫式 Logo（請自行放入）
└── README.md
```

## 🚀 部署到 GitHub Pages

1. 在 GitHub 建立新的 Repository（例如：`xinglongaplus.github.io`）
2. 把這個資料夾的所有檔案上傳
3. 到 Settings → Pages → 選擇 `main` branch
4. 等幾分鐘後就可以用 `https://你的帳號.github.io` 訪問了

## 🖼️ Logo 放置

請將你的兩個 Logo PNG 檔案放入 `assets/` 資料夾：
- `logo-vertical.png` — 直式（用於首頁大 Logo 和 Footer）
- `logo-horizontal.png` — 橫式（用於導航列）

## ✨ 功能

- 精緻開場動畫（約 3.5 秒）
- 浮動顏料色塊背景
- 滾動動畫效果
- 響應式設計（支援手機/平板/電腦）
